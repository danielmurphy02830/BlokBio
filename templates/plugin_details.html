Workflow Description

Anomaly Detection AI utilizes an ensemble of Isolation Forests and Robust PCA to identify problematic samples in bulk RNA-seq datasets. It is designed to work seamlessly with Delta's core processing engine, allowing for real-time quality control during the ingestion phase.

  • check_circle Zero-configuration setup for raw counts
  • check_circle Optimized for serverless AWS Lambda
  • check_circle Interactive PCA & Heatmap outliers
  • check_circle Exportable JSON reports for auditors
